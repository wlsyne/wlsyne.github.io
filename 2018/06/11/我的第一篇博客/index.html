MarkDown语法总结

Syne 6月 11, 2018
  • markdown语法
  • 分享到微博
  • 分享到 Twitter
  • 分享到 Facebook
  • 分享到 Google+

这是我常用的MarkDown语法

1.斜体字与加粗

1.1 斜体字

语法：* *
代码: 

*一支穿云箭*

演示效果：一支穿云箭

1.2 加粗字

语法：** **
代码: 

  **千军万马来相见**

演示效果：千军万马来相见

2.删除线

语法：~~ ~~
代码: 

 ~~问你苦不苦？~~

演示效果：问你苦不苦？

3.代码块

语法：``` ```
代码: 

``` html
<div class="xixi" id="haha"><p>我什么都不知道啊</p></div>
<p>你猜我知不知道</p>
```

演示效果：

<div class="xixi" id="haha"><p>我什么都不知道啊</p></div>
<p>你猜我知不知道</p>

PS:加在```后的是语言类型，如：javascript、html、css等

4.链接

4.1地址链接

语法：[title](链接地址)
代码：

[百度一下](http://www.taobao.com)

演示效果：百度一下

4.12邮箱链接

语法：<邮箱地址>
代码：

<738481015@qq.com>

演示效果：738481015@qq.com

5.图片

5.1插入普通图片

语法：![图片alt](图片地址)
代码：

![测试一下图片](./bg1.jp代码)

演示效果：

5.2插入带链接的图片

语法：[![图片alt](图片地址)](链接地址)
代码：

[![测试一下图片](./bg1.jp代码)](http://www.baidu.com)

演示效果：

6.列表

6.1有序列表

语法：1. 2. 3.
代码：

1. 嘻嘻嘻
2. 哈哈哈
3. 呵呵呵

演示效果：

  1. 嘻嘻嘻
  2. 哈哈哈
  3. 呵呵呵

PS：.后面需要加一个空格

6.2无序列表

语法：*
代码：

* 银鞍照白马
* 飒沓如流星

演示效果：

  • 银鞍照白马
  • 飒沓如流星

7.引用

7.1单行引用

语法：>
代码：

> 十步杀一人，千里不留行

演示效果：

十步杀一人，千里不留行

7.2多行引用

语法：>
代码：

> 十步杀一人，千里不留行
> 事了拂衣去，深藏功与名

演示效果：

十步杀一人，千里不留行
事了拂衣去，深藏功与名

7.3多行嵌套

语法：>
代码：

> 云想衣裳花想容，春风拂槛露华浓
>> 若非群玉山头见，会向瑶台月下逢
>>>一枝红艳露凝香，云雨巫山枉断肠

演示效果：

云想衣裳花想容，春风拂槛露华浓

若非群玉山头见，会向瑶台月下逢

一枝红艳露凝香，云雨巫山枉断肠

1.引用块

{% blockquote [author[, source]] [link] [source_link_title] %}
content
{% endblockquote %}
1.1普通引用快

代码：

{% blockquote %}
春江潮水连海平，海上明月共潮生。
滟滟随波千万里，何处春江无月明。
{% endblockquote %}

演示效果：

春江潮水连海平，海上明月共潮生。
滟滟随波千万里，何处春江无月明。

1.2引用书籍的内容

代码：

{% blockquote 《Do not go gentle into that good night》, ——Dylan Thomas %}
Do not go gentle into that good night,
Old age should burn and rave at close of day;
Rage, rage against the dying of the light.
{% endblockquote %}

演示效果：

Do not go gentle into that good night,
Old age should burn and rave at close of day;
Rage, rage against the dying of the light.

《Do not go gentle into that good night》——Dylan Thomas

1.3引用网址的内容

代码：

{% blockquote @Synewl http://synewl.coding.me/ %}
你以为完事啦？
{% endblockquote %}

演示效果：

你以为完事啦？

@Synewl——synewl.coding.me

1.4引用网页上的文章

代码：

{% blockquote Seth Godin http://sethgodin.typepad.com/seths_blog/2009/07/welcome-to-island-marketing.html Welcome to Island Marketing %}
Every interaction is both precious and an opportunity to delight.
{% endblockquote %}

展示效果：

Every interaction is both precious and an opportunity to delight.

Seth GodinWelcome to Island Marketing

2.代码块

{% codeblock [title] [lang:language] [url] [link text] %}
code snippet
{% endcodeblock %}
2.1普通代码块

代码：

{% codeblock %}
console.log('你该不会是个傻子吧？');
{% endcodeblock %}

展示效果：

console.log('你该不会是个傻子吧？');

2.2 指定语言的代码块

代码：

{% codeblock lang:javascript %}
let a={}
let b=[]
{% endcodeblock %}

演示效果：

let a={}
let b=[]
